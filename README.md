# AFRICOCO

AFRICOCO is a Laravel and Filament CMS website for the African Coconut Heritage Initiative.

## Local Preview

Run the local preview server:

```bash
php artisan --env=preview serve --host=127.0.0.1 --port=8010
```

Admin URL:

```text
http://127.0.0.1:8010/admin/login
```
